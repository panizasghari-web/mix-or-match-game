# 🧠 Memory Card Game

A fun and interactive Halloween-themed Memory Card Game built with HTML, Tailwind CSS, and JavaScript. Flip the cards, match every pair before the timer runs out, and challenge yourself to finish with the fewest flips possible!

## ✨ Features

- 🎃 Halloween-inspired UI
- 🎴 Smooth 3D card flip animation
- ⏳ Countdown timer
- 🔄 Flip counter
- 🎯 Match detection system
- ✨ Match animation effects
- 🔒 Prevents invalid or extra clicks
- 🔊 Sound effects for flipping, matching, winning, and losing
- 🏆 Victory screen
- 💀 Game Over screen
- 🎲 Random card shuffle every game
- 🔄 Restart game without refreshing the page
- 📱 Fully responsive design

## 🛠️ Built With

- HTML5
- Tailwind CSS
- JavaScript (ES6)

## 🎮 How to Play

1. Click Start Game to begin.
2. Flip two cards to reveal their icons.
3. If the cards match, they remain open.
4. If they don't match, they'll automatically flip back.
5. Match all pairs before the timer reaches 0.
6. Click the Victory or Game Over screen to start a new game instantly.
